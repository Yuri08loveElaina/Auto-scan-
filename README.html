Elaina Cute - Automated Recon & Vulnerability Scanner

File: elaina-cute.sh

Mô tả

elaina-cute.sh là script tích hợp để thu thập subdomain, kiểm tra endpoint, scan payload và tạo báo cáo tương tác.

Sử dụng các công cụ: Subfinder, Httpx, Katana, GauPlus, Selenium, Jinja2, Pandas/XlsxWriter.

Payload thử nghiệm: XSS, SQLi, LFI, RCE, SSRF, SSTI, XXE, redirect, CRLF, cache, JWT, JSON, file upload, template injection.

Kết quả ghi nhận: server info, severity, AI risk score, CVEs, screenshot payload.

Yêu cầu

  • OS: Linux / Termux / WSL / macOS
  • Python 3.x
  • Go >= 1.20
  • Chrome/Chromium + Chromedriver
  • pip packages: requests, beautifulsoup4, selenium, pandas, xlsxwriter, jinja2, pillow, dash, dash_table, dash_html_components, dash_core_components

Cách dùng

Cấp quyền:

chmod +x elaina-cute.sh

Chạy scan:

./elaina-cute.sh domain1.com domain2.com

Ví dụ:

./elaina-cute.sh example.com testsite.org

Quy trình hoạt động

  1. Kiểm tra & cài Go nếu chưa có
  2. Cài package Python
  3. Cài các công cụ: subfinder, httpx, katana, gauplus
  4. Thu thập subdomain bằng subfinder
  5. Kiểm tra live host bằng httpx
  6. Thu thập endpoint bằng katana
  7. Thu thập URL lịch sử bằng gauplus
  8. Chạy Python script:
    • Thử payload injection trên endpoint
    • Lưu kết quả: server, severity, AI risk score, CVEs
    • Chụp screenshot payload
  9. Xuất báo cáo:
    • Excel: report.xlsx
    • HTML tương tác: report_interactive.html
    • Screenshot: thư mục screenshots/

Output

  • Excel report: report.xlsx
  • HTML interactive report: report_interactive.html
  • Screenshots: thư mục screenshots/
